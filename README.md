# Notes App

A **fully responsive web-based Notes App** built using **HTML, CSS, and JavaScript**. This application allows users to efficiently **add, view, search, and delete notes**, with support for multiple notes and sub-notes. The app is designed to be user-friendly and works seamlessly across **desktop, tablet, and mobile devices**.

---

## Features

### 1. Add Notes
- Easily add new notes using a **sliding input box**.
- Supports **multiple lines** of text.
- Each note can hold **sub-notes**, allowing you to organize information hierarchically.

### 2. View Notes
- All created notes are displayed on the **home page** in a clean card layout.
- Notes automatically adjust to screen size, making it readable on all devices.

### 3. Delete Notes
- Remove any note that is no longer needed using a **delete button** on each note.
- Helps keep your workspace organized.

### 4. Search Notes
- **Real-time search bar** filters notes as you type.
- Quickly find notes based on content or keywords.

### 5. Responsive Design
- Works perfectly on **desktop, tablet, and mobile screens**.
- Notes stack vertically on smaller devices, maintaining readability.
- Input boxes and buttons adjust automatically for different screen sizes.

### 6. Clean UI
- Simple, modern, and minimalistic design.
- Focus on usability and readability.
